Flex

The 'flex' style consists of 3 properties:

Flex Grow Flex Shrink Flex Basis 
.flex-item{
    flex: flex-grow flex-shrink flex-basis;
    flex:     1          1        100px   ;
}

1

2

3

4


In this example I have applied [0 grow, 0 shrink, 100px basis].
These means the items will not grow to the container, will not shrink below the item's 'flex-basis' size, and have a default size of 100px; 

.flex-item{
    flex: 0 0 100px;
}

1

2

3

4


In this example I have applied [1 grow, 0 shrink, 100px basis].
These means the items will grow to the container, will not shrink below the item's 'flex-basis' size, and have a default size of 100px; 

.flex-item{
    flex: 1 0 100px;
}

In this example I have applied [0 grow, 1 shrink, 100px basis].
These means the items will not grow to the container, will shrink to the container, and have a default size of 100px; 

.flex-item{
    flex: 0 1 100px;
}

Learn more about the properties of 'flex' below.

Flex Grow Flex Shrink Flex Basis
Flex Grow

In this example setting a value to 'flex-grow' will force the 'flex-items' to fill their 'flex-container'. 

.flex-item{
    flex-grow: 1;
}

1

2

3


Increasing item2's flex-grow to 3 hasn't tripled it's size.
Item2 instead has a size that equals 3/5's of it's container.
It's container total 'flex-grow' size being a compination of all of the 'flex-grow' values added together.
1 from item1, 3 from item2, and 1 from item3. (1 + 3 + 1 = 5) 

.flex-item{
    flex-grow: 1;
}
.flex-item.item2{
    flex-grow: 3;
}

Flex Shrink

By setting 'flex-shrink' to '1' all flex-items decrease in size at the same rate. 

.flex-item{
    flex-basis: 1000px;
    flex-shrink: 1;
}

1

2

3


By setting 'flex-shrink' to '0' no flex-items will decrease in size. 

.flex-item{
    flex-basis: 200px;
    flex-shrink: 0;
}

1

2

3


Adding a 'flex-grow' style will remove the whitespace from the container while the 'flex-shrink' still prevents the items from being below 200px; 

.flex-item{
    flex-basis: 200px;
    flex-shrink: 0;
    flex-grow: 1;
}
.flex-item.item3{
    flex-shrink: 3;
}

1

2

3


By setting item1's 'flex-shrink' to '1' and item2's 'flex-shrink' to '2', item2 will decrease in size faster than item1.
However it is important to note, item2 will not be twice as small as item1. 

.flex-item{
    flex-basis: 500px;
    flex-shrink: 1;
}
.flex-item.item2{
    flex-shrink: 2;
}

Flex Basis

Flex basis can either be the width of the 'flex-items', when the 'flex-container' is a 'row' or height of the 'flex-items', when the 'flex-container' is a 'column'. 

.flex-container{
    flex-direction: row;
}
.flex-item{
    flex-basis: 100px;
}
